# Seocho-animallove
팀프로젝트-서초동물사랑센터-웹사이트
서초동물사랑센터는 버려진 유기견들을 보호하고 유기견들을 분양해 새로운 주인을 찾을수 있게끔 목적을 가진 사이트입니다.<br>
기존 사이트는 레이아웃 구성 자체가 좁은 콘텐츠 구역에 너무 많은 정보를 담아 복잡함을 주고 있어 좀 더 간결하게 구성해보았습니다.
